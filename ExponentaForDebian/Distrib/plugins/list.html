rdesktop-1.4.1.tar.gz -- дострибутив RDesktop — GUI RDP клиента для Linux

Данная версия используется для совместимости с программами подбора паролей (смотри патч rdp-brute-force-r422.diff) Установка патча на дистрибутив следующая:

1. Распакуйте дистрибутив в папку

2. В корне папки выполните команду:

# patch -p1 -i rdp-brute-force-r422.diff

3. Установите пакет, как это обычно делается из архива исходных кодов

После этого Ваш клиент готов к брутфорсу. Ссылка: https://www.ethicalhacker.net/columns/gates/tutorial-ms-terminal-server-cracking
