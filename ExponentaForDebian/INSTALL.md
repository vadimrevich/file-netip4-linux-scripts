Установка
========

1. Создайте файл `Exponenta1-Main-Install.tar.gz` и поместите его на сервер. (см.INSTALL.md в каталоге opt.tmp.Exponenta1)
2. Скачайте на Linux компьютер файл loadmaindistrib.sh для обычной установки или loadhiddenmaindistrib.sh для скрытой установки админ-пака
3. Запустите его с правами суперпользователя.

Файлы testmaindistrib.sh и testhiddenmaindistrib.sh приведены для тестирования установки в автономном режиме, когда архив располагается в том же каталоге, что и установочный файл.
