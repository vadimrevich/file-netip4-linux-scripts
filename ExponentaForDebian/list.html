Содержимое каталога

../index.html — Заглушка корня сайта

index.html — Заглушка каталога

descript.ion -- Описание каталога (на русском языке)

opt.tmp.Exponenta1/ -- каталог с файлами инсталлятора

loadmaindistrib.sh -- загрузка и установка модулей админ-пака по http протоколу (консольный вариант, видимая установка)

Это специфичный файл для каждого сервера инсталляции админ-пака! Поставьте в него корректные значения!

Загрузка админ-пера может осуществляться по http, https и ftp прортоколуюДля этого нужно только поменять переменные в файле. В будущих версиях добавится поддержка опций curl/wget

loadhiddenmaindistrib.sh -- загрузка и установка модулей админ-пака по http протоколу (консольный вариант, скрытая установка)

Это специфичный файл для каждого сервера скрытой инсталляции админ-пака! Поставьте в него корректные значения!

Загрузка админ-пера может осуществляться по http, https и ftp прортоколуюДля этого нужно только поменять переменные в файле. В будущих версиях добавится поддержка опций curl/wget

INSTALL.md — Файл с описание подготовки инсталлятора для установки Админ-пака. Файл имеет формат GitHub Markdown, читается и просматривается специальными программами и любым текстовым редактором

Distrib/plugins/ — каталог с файлами дополнительных дистрибутивов, отсутствующих в Ubuntu и загружаемых при установке админ-пака в каталог $Exponenta1/Distrib/plugins. ДЛя просмотра содержимого перейдите по ссылке

Дополнительные ссылки:

Exponenta/ — каталог с админ-паком Exponenta for Windows на этом сервере

ExponentaForDebian — Каталог с файлами Exponenta for Linux на оригинальном сервере
